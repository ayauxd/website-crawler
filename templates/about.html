Tips for Developers

The archived site can be a great starting point for your own projects:

  • Use the HTML structure as a template for your own pages
  • Study the CSS to understand the styling techniques used
  • Examine JavaScript code to learn interactive features
  • Modify and extend the code to create your own version

About Serverless Architecture

Crawlr uses a serverless architecture which means:

  • The crawler runs in short bursts (10-second time limit)
  • Larger sites require multiple "Continue Crawl" requests
  • Files are stored temporarily and may be cleaned up after a period of time
  • For ongoing archiving, consider downloading the files
